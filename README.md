# Практична робота 7.1 — Варіант 1
## Система управління студентами

### Опис
Веб-застосунок для управління списком студентів з фільтрацією, сортуванням та аналізом даних.

### Функціонал
- Додавання студентів (ім'я, вік, середній бал, факультет)
- Фільтрація за ім'ям, факультетом, мінімальним балом, максимальним віком
- Сортування за ім'ям, віком, балом (ASC/DESC)
- Статистика: середній бал, найкращий студент, розподіл за факультетами
- Пошук за ім'ям з підсвіткою результатів
- Валідація вхідних даних

### Використані методи масивів
- `filter()` — фільтрація студентів за критеріями
- `sort()` — сортування за різними полями
- `find()` / `reduce()` — статистика та пошук найкращого

### Як запустити
1. Завантажити файл `students-7-1.html`
2. Відкрити у браузері (подвійний клік)
3. Не потребує встановлення залежностей

### Технології
Vanilla JS (ES6+), HTML5, CSS3. Без зовнішніх бібліотек.
